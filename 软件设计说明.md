# TOPSIS作战评估系统软件设计说明

## 1. 系统概述

### 1.1 系统目标
TOPSIS作战评估系统是一个基于多准则决策分析的作战效能评估工具，使用TOPSIS（Technique for Order Preference by Similarity to Ideal Solution）算法对四种作战场景进行量化评估。

### 1.2 核心功能
- 四种作战场景的参数化建模
- 基于TOPSIS算法的效能评估
- 可视化结果展示
- 智能化改进建议生成

### 1.3 技术架构
- **前端框架**: Streamlit 1.28.0+
- **数值计算**: NumPy 1.21.0+
- **数据处理**: Pandas 1.3.0+
- **可视化**: Plotly 5.0.0+
- **配置管理**: YAML 6.0+

## 2. 详细算法设计

### 2.1 TOPSIS算法原理

#### 2.1.1 算法流程
1. **决策矩阵构建**: 将参数输入转换为标准化决策矩阵
2. **权重分配**: 根据作战重要性分配指标权重
3. **标准化处理**: 消除量纲影响，统一指标尺度
4. **加权决策矩阵**: 计算加权标准化决策矩阵
5. **理想解确定**: 找到正理想解和负理想解
6. **距离计算**: 计算各方案到理想解的距离
7. **相对贴近度**: 计算综合评价指数
8. **方案排序**: 根据相对贴近度进行排序

#### 2.1.2 数学模型

**标准化公式**:
```
r_ij = x_ij / √(∑ᵢ₌₁ⁿ x_ij²)
其中：
- r_ij: 标准化后的值
- x_ij: 原始参数值
- n: 方案数量
- i: 方案索引，j: 指标索引
```

**加权标准化矩阵**:
```
v_ij = w_j × r_ij
其中：
- v_ij: 加权标准化值
- w_j: 第j个指标的权重
- r_ij: 标准化值
```

**欧几里得距离**:
```
D⁺_i = √[∑ⱼ₌₁ᵐ (v_ij - v⁺_j)²]
D⁻_i = √[∑ⱼ₌₁ᵐ (v_ij - v⁻_j)²]
其中：
- D⁺_i: 第i个方案到正理想解的距离
- D⁻_i: 第i个方案到负理想解的距离
- v⁺_j: 正理想解的第j个分量
- v⁻_j: 负理想解的第j个分量
- m: 指标数量
```

**相对贴近度**:
```
C_i = D⁻_i / (D⁺_i + D⁻_i)
其中：
- C_i: 第i个方案的相对贴近度
- C_i ∈ [0,1]，值越大方案越优
```

### 2.2 指标体系详细设计

#### 2.2.1 五大核心指标定义

1. **综合作战能力 (Overall Combat Power)**
   - **定义**: 反映作战力量的整体水平，包括火力、机动、防护等综合能力
   - **计算公式**:
   ```
   OCP = 0.35 × FP + 0.25 × MC + 0.20 × PC + 0.20 × IC
   其中：
   - FP: 火力打击能力 (Firepower)
   - MC: 机动能力 (Mobility Capability)
   - PC: 防护能力 (Protection Capability)
   - IC: 信息化能力 (Information Capability)
   ```

2. **作战效能 (Operational Effectiveness)**
   - **定义**: 作战任务的执行效率，包括目标完成率、资源利用率等
   - **计算公式**:
   ```
   OE = 0.30 × CR + 0.25 × RU + 0.25 × TE + 0.20 × PA
   其中：
   - CR: 目标完成率 (Completion Rate)
   - RU: 资源利用率 (Resource Utilization)
   - TE: 时间效率 (Time Efficiency)
   - PA: 精确打击能力 (Precision Accuracy)
   ```

3. **作战适应性 (Combat Adaptability)**
   - **定义**: 适应不同作战环境的能力，包括环境适应性、目标适应性等
   - **计算公式**:
   ```
   CA = 0.30 × EA + 0.25 × TA + 0.25 × TF + 0.20 × AR
   其中：
   - EA: 环境适应性 (Environmental Adaptability)
   - TA: 目标适应性 (Target Adaptability)
   - TF: 战术灵活性 (Tactical Flexibility)
   - AR: 抗干扰能力 (Anti-interference Resilience)
   ```

4. **作战保障 (Logistical Support)**
   - **定义**: 后勤保障能力，包括弹药、油料、维修、医疗等保障
   - **计算公式**:
   ```
   LS = 0.30 × AS + 0.25 × FS + 0.25 × MS + 0.20 × MS
   其中：
   - AS: 弹药保障 (Ammunition Support)
   - FS: 油料保障 (Fuel Support)
   - MS: 维修保障 (Maintenance Support)
   - MS: 医疗保障 (Medical Support)
   ```

5. **协同能力 (Coordination Capability)**
   - **定义**: 多军兵种协同作战能力，包括部队协同、兵种协同等
   - **计算公式**:
   ```
   CC = 0.30 × FC + 0.25 × SC + 0.25 × CC + 0.20 × IC
   其中：
   - FC: 部队协同 (Force Coordination)
   - SC: 兵种协同 (Service Coordination)
   - CC: 指挥协同 (Command Coordination)
   - IC: 情报协同 (Intelligence Coordination)
   ```

#### 2.2.2 指标量化详细方法

**1. 线性标准化法**:
```
对于正向指标：
x'_ij = (x_ij - min(x_j)) / (max(x_j) - min(x_j))

对于负向指标：
x'_ij = (max(x_j) - x_ij) / (max(x_j) - min(x_j))

其中：
- x'_ij: 标准化后的指标值
- x_ij: 原始指标值
- max(x_j): 第j个指标的最大值
- min(x_j): 第j个指标的最小值
```

**2. 向量标准化法**:
```
r_ij = x_ij / √(∑ᵢ₌₁ⁿ x_ij²)

特点：
- 保留原始数据的分布特征
- 适用于所有指标为正向或负向的情况
- 消除量纲影响
```

**3. 分段标准化法**:
```
对于具有阈值特征的指标：
x'_ij = ⎧
        ⎪ 0.1,                    x_ij ≤ T₁
        ⎪ 0.1 + 0.4 × (x_ij - T₁)/(T₂ - T₁),  T₁ < x_ij ≤ T₂
        ⎪ 0.5 + 0.4 × (x_ij - T₂)/(T₃ - T₂),  T₂ < x_ij ≤ T₃
        ⎪ 0.9 + 0.1 × (x_ij - T₃)/(T₄ - T₃),  T₃ < x_ij ≤ T₄
        ⎪ 1.0,                    x_ij > T₄
        ⎩

其中：T₁, T₂, T₃, T₄ 为阈值点
```

#### 2.4.1 参数到指标映射

```

#### 2.4.2 指标综合计算

**第一步：计算参数贡献值**
```
对于每个参数p，计算其对各个指标的贡献值：
C_p,i = w_p,i × Q_p

其中：
- C_p,i: 参数p对指标i的贡献值
- w_p,i: 参数p对指标i的权重
- Q_p: 参数p的量化值
```

**第二步：计算指标值**
```
对于每个指标i，综合所有参数的贡献：
I_i = min(1.0, ∑ C_p,i)

其中：
- I_i: 指标i的最终值
- C_p,i: 所有对该指标有贡献的参数值
- 限制最大值为1.0，避免过度累积
```

**第三步：TOPSIS评估**
```
1. 构建决策矩阵X = [I_1, I_2, I_3, I_4, I_5]
2. 标准化：r_ij = x_ij / √(∑ᵢ x_ij²)
3. 加权：v_ij = w_j × r_ij
4. 确定理想解：
   v⁺ = [max(v_1), max(v_2), max(v_3), max(v_4), max(v_5)]
   v⁻ = [min(v_1), min(v_2), min(v_3), min(v_4), min(v_5)]
5. 计算距离：
   D⁺ = √[∑ⱼ (v_j - v⁺_j)²]
   D⁻ = √[∑ⱼ (v_j - v⁻_j)²]
6. 计算相对贴近度：
   C = D⁻ / (D⁺ + D⁻)
```

#### 2.4.3 最终得分计算

**综合得分公式**:
```
Final_Score = C = D⁻ / (D⁺ + D⁻)

其中：
- Final_Score: 最终得分 ∈ [0,1]
- D⁺: 到正理想解的距离
- D⁻: 到负理想解的距离
```

**等级划分标准**:
```
A级（优秀）：0.8 ≤ Score ≤ 1.0
B级（良好）：0.6 ≤ Score < 0.8
C级（一般）：0.4 ≤ Score < 0.6
D级（较差）：0.0 ≤ Score < 0.4
```

### 2.6 量化方式选择理由

#### 2.6.1 分段线性函数

**应用场景**: 数量型参数（舰艇、飞机、人员等）
**选择理由**:
- 符合作战中资源投入的边际效益递减规律
- 避免简单线性函数的无限制增长
- 体现资源配置的最优平衡点
- 便于设置合理的阈值和极限值

**数学特征**:
```
f(x) = ⎧
       ⎪ a + b×x,              x ≤ x₁
       ⎪ c + d×x,              x₁ < x ≤ x₂
       ⎪ e + f×x,              x₂ < x ≤ x₃
       ⎩
其中：b > d > f，体现边际效益递减
```

#### 2.6.2 对数函数

**应用场景**: 距离型参数（导弹射程、通信距离等）
**选择理由**:
- 距离增长的战略价值呈对数递减
- 符合作战半径的实际意义
- 避免距离过大导致的数值失真
- 体现距离增加的边际成本递增

**数学特征**:
```
f(x) = a + b×log(x)
其中：b > 0，函数单调递增但增速递减
```

#### 2.6.3 阶梯函数

**应用场景**: 等级型参数（系统等级、技术等级等）
**选择理由**:
- 技术等级具有质变性，非连续变化
- 体现技术进步的跨越式特征
- 符合实际技术发展规律
- 便于分级管理和评估

**数学特征**:
```
f(x) = ⎧
       ⎪ v₁,    x ∈ [l₁, u₁]
       ⎪ v₂,    x ∈ [l₂, u₂]
       ⎪ v₃,    x ∈ [l₃, u₃]
       ⎩
其中：v₁ < v₂ < v₃，体现等级提升
```

#### 2.6.4 线性插值法

**应用场景**: 连续型参数的处理
**选择理由**:
- 保持数据的连续性
- 避免跳跃式变化
- 提高计算精度
- 便于处理非整数值

**数学特征**:
```
f(x) = f(x₁) + (x - x₁)×(f(x₂) - f(x₁))/(x₂ - x₁)
其中：x₁ ≤ x ≤ x₂
```

```python
class TOPSISEvaluator:
    """TOPSIS评估器核心类"""

    def __init__(self, weights: Dict[str, float]):
        """
        初始化TOPSIS评估器

        Args:
            weights: 指标权重字典
        """
        self.weights = weights
        self.indicators = list(weights.keys())
        self.normalized_matrix = None
        self.weighted_matrix = None
        self.ideal_solution = None
        self.negative_ideal_solution = None

    def normalize_matrix(self, decision_matrix: np.ndarray) -> np.ndarray:
        """
        标准化决策矩阵

        Args:
            decision_matrix: 原始决策矩阵

        Returns:
            标准化后的决策矩阵
        """
        # 计算每列的平方和的平方根
        norm_factors = np.sqrt(np.sum(decision_matrix ** 2, axis=0))
        # 避免除零错误
        norm_factors = np.where(norm_factors == 0, 1, norm_factors)
        # 标准化
        normalized = decision_matrix / norm_factors
        return normalized

    def calculate_weighted_matrix(self, normalized_matrix: np.ndarray) -> np.ndarray:
        """
        计算加权标准化矩阵

        Args:
            normalized_matrix: 标准化矩阵

        Returns:
            加权标准化矩阵
        """
        # 转换权重为numpy数组
        weight_array = np.array([self.weights[indicator] for indicator in self.indicators])
        # 计算加权矩阵
        weighted = normalized_matrix * weight_array
        return weighted

    def find_ideal_solutions(self, weighted_matrix: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
        """
        确定理想解和负理想解

        Args:
            weighted_matrix: 加权标准化矩阵

        Returns:
            正理想解和负理想解
        """
        # 确定每个指标的最佳和最差值
        ideal_solution = np.max(weighted_matrix, axis=0)
        negative_ideal_solution = np.min(weighted_matrix, axis=0)
        return ideal_solution, negative_ideal_solution

    def calculate_distances(self, weighted_matrix: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
        """
        计算到理想解的距离

        Args:
            weighted_matrix: 加权标准化矩阵

        Returns:
            到正理想解和负理想解的距离
        """
        # 计算到正理想解的距离
        positive_distances = np.sqrt(np.sum((weighted_matrix - self.ideal_solution) ** 2, axis=1))
        # 计算到负理想解的距离
        negative_distances = np.sqrt(np.sum((weighted_matrix - self.negative_ideal_solution) ** 2, axis=1))
        return positive_distances, negative_distances

    def calculate_similarity(self, positive_distances: np.ndarray, negative_distances: np.ndarray) -> np.ndarray:
        """
        计算相对贴近度

        Args:
            positive_distances: 到正理想解的距离
            negative_distances: 到负理想解的距离

        Returns:
            相对贴近度数组
        """
        # 避免除零错误
        total_distances = positive_distances + negative_distances
        total_distances = np.where(total_distances == 0, 1e-10, total_distances)
        # 计算相对贴近度
        similarity = negative_distances / total_distances
        return similarity

    def evaluate(self, scenario_data: Dict[str, float]) -> Dict[str, float]:
        """
        执行TOPSIS评估

        Args:
            scenario_data: 场景数据

        Returns:
            评估结果字典
        """
        # 1. 构建决策矩阵
        decision_matrix = self._build_decision_matrix(scenario_data)

        # 2. 标准化处理
        self.normalized_matrix = self.normalize_matrix(decision_matrix)

        # 3. 计算加权标准化矩阵
        self.weighted_matrix = self.calculate_weighted_matrix(self.normalized_matrix)

        # 4. 确定理想解
        self.ideal_solution, self.negative_ideal_solution = self.find_ideal_solutions(self.weighted_matrix)

        # 5. 计算距离
        positive_distances, negative_distances = self.calculate_distances(self.weighted_matrix)

        # 6. 计算相对贴近度
        similarity_scores = self.calculate_similarity(positive_distances, negative_distances)

        # 7. 返回结果
        return self._format_results(similarity_scores)
```

```python
def _quantify_parameters(self, raw_params: Dict[str, float]) -> Dict[str, float]:
    """
    量化原始参数为指标值

    Args:
        raw_params: 原始参数字典

    Returns:
        量化后的指标值字典
    """
    quantified = {}

    for param_name, param_value in raw_params.items():
        if param_name in self.parameter_mapping:
            mapping = self.parameter_mapping[param_name]
            indicator_name = mapping['indicator']

            # 根据参数类型采用不同的量化方法
            if param_name in ['ship_count', 'submarine_count', 'aircraft_count']:
                # 数量型参数：采用分段线性函数
                quantified[indicator_name] = self._quantify_count_parameter(param_value, param_name)
            elif param_name in ['missile_range', 'communication_range']:
                # 距离型参数：采用对数函数
                quantified[indicator_name] = self._quantify_range_parameter(param_value)
            elif param_name in ['communication_system', 'electronic_warfare']:
                # 系统型参数：采用阶梯函数
                quantified[indicator_name] = self._quantify_system_parameter(param_value)
            else:
                # 其他参数：采用线性归一化
                quantified[indicator_name] = self._quantify_linear_parameter(param_value, param_name)

    return quantified

def _quantify_count_parameter(self, count: float, param_type: str) -> float:
    """
    量化数量型参数

    采用分段线性函数，考虑边际效益递减原则

    Args:
        count: 参数数量
        param_type: 参数类型

    Returns:
        量化值 (0-1之间)
    """
    # 定义不同类型参数的基准值
    benchmarks = {
        'ship_count': {'min': 1, 'max': 50, 'optimal': 30},
        'submarine_count': {'min': 1, 'max': 20, 'optimal': 12},
        'aircraft_count': {'min': 1, 'max': 100, 'optimal': 60}
    }

    if param_type not in benchmarks:
        return min(count / 50, 1.0)  # 默认基准

    benchmark = benchmarks[param_type]

    # 采用梯形隶属函数
    if count <= benchmark['min']:
        return 0.1
    elif count <= benchmark['optimal']:
        # 上升段：线性增长
        return 0.1 + 0.8 * (count - benchmark['min']) / (benchmark['optimal'] - benchmark['min'])
    elif count <= benchmark['max']:
        # 平稳段：缓慢增长
        return 0.9 + 0.1 * (count - benchmark['optimal']) / (benchmark['max'] - benchmark['optimal'])
    else:
        return 1.0

def _quantify_range_parameter(self, range_value: float) -> float:
    """
    量化距离型参数

    采用对数函数，体现距离增长的战略价值

    Args:
        range_value: 距离值

    Returns:
        量化值 (0-1之间)
    """
    # 定义基准距离
    min_range = 10   # 最小有效距离
    max_range = 1000 # 最大有效距离

    # 采用对数函数进行量化
    if range_value <= min_range:
        return 0.1
    elif range_value >= max_range:
        return 1.0
    else:
        # 对数函数：log(x)/log(max)
        import math
        normalized = math.log(range_value) / math.log(max_range)
        return 0.1 + 0.9 * normalized

def _quantify_system_parameter(self, system_level: float) -> float:
    """
    量化系统型参数

    采用阶梯函数，体现系统集成的价值

    Args:
        system_level: 系统水平 (1-10)

    Returns:
        量化值 (0-1之间)
    """
    # 定义系统水平对应的价值
    system_values = {
        1: 0.1,   # 基础系统
        2: 0.2,   # 改进系统
        3: 0.3,   # 标准系统
        4: 0.4,   # 先进系统
        5: 0.5,   # 现代化系统
        6: 0.65,  # 信息化系统
        7: 0.75,  # 网络化系统
        8: 0.85,  # 智能化系统
        9: 0.95,  # 先进智能化系统
        10: 1.0   # 完全智能化系统
    }

    # 线性插值处理非整数等级
    lower_level = int(system_level)
    upper_level = min(lower_level + 1, 10)

    if lower_level == upper_level:
        return system_values[lower_level]
    else:
        # 线性插值
        weight = system_level - lower_level
        lower_value = system_values[lower_level]
        upper_value = system_values[upper_level]
        return lower_value + weight * (upper_value - lower_value)

class ParameterValidator:
    """参数验证器"""

    def __init__(self, config_path: str = "config/scenarios.yaml"):
        """
        初始化参数验证器

        Args:
            config_path: 配置文件路径
        """
        self.config = self._load_config(config_path)
        self.validation_rules = self._build_validation_rules()

    def validate_parameters(self, scenario_type: str, parameters: Dict[str, float]) -> Tuple[bool, List[str]]:
        """
        验证参数的有效性

        Args:
            scenario_type: 场景类型
            parameters: 参数字典

        Returns:
            (是否有效, 错误信息列表)
        """
        errors = []

        # 检查场景类型是否有效
        if scenario_type not in self.config:
            errors.append(f"无效的场景类型: {scenario_type}")
            return False, errors

        # 检查必需参数
        required_params = set(self.config[scenario_type]['parameters'].keys())
        provided_params = set(parameters.keys())
        missing_params = required_params - provided_params

        if missing_params:
            errors.append(f"缺少必需参数: {', '.join(missing_params)}")

        # 检查参数值范围
        for param_name, param_value in parameters.items():
            if param_name in self.validation_rules[scenario_type]:
                rule = self.validation_rules[scenario_type][param_name]

                # 检查类型
                if not isinstance(param_value, (int, float)):
                    errors.append(f"参数 {param_name} 必须为数值类型")
                    continue

                # 检查范围
                if param_value < rule['min_value']:
                    errors.append(f"参数 {param_name} 值 {param_value} 小于最小值 {rule['min_value']}")
                elif param_value > rule['max_value']:
                    errors.append(f"参数 {param_name} 值 {param_value} 大于最大值 {rule['max_value']}")

                # 检查特殊约束
                if 'constraints' in rule:
                    for constraint in rule['constraints']:
                        if not self._check_constraint(param_value, constraint):
                            errors.append(f"参数 {param_name} 不满足约束条件: {constraint['description']}")

        return len(errors) == 0, errors

    def _check_constraint(self, value: float, constraint: Dict) -> bool:
        """
        检查特殊约束条件

        Args:
            value: 参数值
            constraint: 约束条件

        Returns:
            是否满足约束
        """
        constraint_type = constraint['type']

        if constraint_type == 'multiple_of':
            return value % constraint['value'] == 0
        elif constraint_type == 'greater_than':
            return value > constraint['value']
        elif constraint_type == 'less_than':
            return value < constraint['value']
        elif constraint_type == 'in_range':
            return constraint['min'] <= value <= constraint['max']
        else:
            return True
```

### 4.3 结果输出设计

```python
def format_results(self, similarity_scores: np.ndarray) -> Dict[str, Any]:
    """
    格式化评估结果

    Args:
        similarity_scores: 相对贴近度数组

    Returns:
        格式化的结果字典
    """
    # 获取当前方案得分
    current_score = similarity_scores[2]  # 第三个是当前方案

    # 确定等级
    if current_score >= 0.8:
        grade = "优秀"
        level = "A"
    elif current_score >= 0.6:
        grade = "良好"
        level = "B"
    elif current_score >= 0.4:
        grade = "一般"
        level = "C"
    else:
        grade = "较差"
        level = "D"

    # 计算排名
    ranking = np.argsort(similarity_scores)[::-1]
    current_rank = np.where(ranking == 2)[0][0] + 1

    # 生成建议
    suggestions = self._generate_suggestions(current_score)

    return {
        'score': current_score,
        'grade': grade,
        'level': level,
        'ranking': current_rank,
        'total_alternatives': len(similarity_scores),
        'suggestions': suggestions,
        'detailed_scores': {
            'current': current_score,
            'excellent': similarity_scores[0],
            'good': similarity_scores[1],
            'average': similarity_scores[3],
            'poor': similarity_scores[4]
        }
    }
```

```python
class TOPSISWebApp:
    """TOPSIS Web应用主类"""

    def __init__(self):
        """初始化Web应用"""
        self.evaluator = TOPSISEvaluator(self._load_weights())
        self.validator = ParameterValidator()
        self.visualizer = VisualizationGenerator()

    def run(self):
        """运行Web应用"""
        # Streamlit UI逻辑
        st.set_page_config(
            page_title="TOPSIS作战评估系统",
            page_icon="⚔️",
            layout="wide"
        )

        # 侧边栏：场景选择
        self._render_sidebar()

        # 主界面：参数输入
        self._render_parameter_input()

        # 评估按钮
        if st.button("开始评估", type="primary"):
            self._handle_evaluation()

    def _render_sidebar(self):
        """渲染侧边栏"""
        st.sidebar.title("🎯 作战场景选择")

        # 场景选择
        scenario_options = {
            'maritime_blockade': '海上封锁作战',
            'landing_zone_clearance': '登陆场清扫作战',
            'underwater_surveillance': '水下监视作战',
            'mine_countermeasures': '水雷清排作战'
        }

        selected_scenario = st.sidebar.selectbox(
            "选择作战场景",
            options=list(scenario_options.keys()),
            format_func=lambda x: scenario_options[x]
        )

        st.session_state.current_scenario = selected_scenario

        # 显示场景描述
        descriptions = {
            'maritime_blockade': '评估海上封锁作战方案的效能',
            'landing_zone_clearance': '评估登陆场清扫作战方案的效能',
            'underwater_surveillance': '评估水下监视作战方案的效能',
            'mine_countermeasures': '评估水雷清排作战方案的效能'
        }

        st.sidebar.info(descriptions[selected_scenario])

    def _render_parameter_input(self):
        """渲染参数输入界面"""
        st.header("⚙️ 作战参数配置")

        # 获取参数信息
        param_info = self.validator.get_parameter_info(st.session_state.current_scenario)

        # 创建参数输入表单
        with st.form("parameter_form"):
            for param_name, param_data in param_info.items():
                col1, col2 = st.columns([1, 2])

                with col1:
                    st.write(f"**{param_data['name']}**")
                    st.caption(param_data['description'])

                with col2:
                    if param_data['type'] == 'integer':
                        value = st.number_input(
                            f"{param_data['name']} ({param_data['unit']})",
                            min_value=param_data['min_value'],
                            max_value=param_data['max_value'],
                            value=param_data['default_value'],
                            step=1,
                            key=param_name
                        )
                    else:
                        value = st.number_input(
                            f"{param_data['name']} ({param_data['unit']})",
                            min_value=param_data['min_value'],
                            max_value=param_data['max_value'],
                            value=param_data['default_value'],
                            step=0.1,
                            key=param_name
                        )

                    st.session_state.parameters[param_name] = float(value)

            submitted = st.form_submit_button("提交参数")

            if submitted:
                self._validate_and_evaluate()

    def _handle_evaluation(self):
        """处理评估请求"""
        try:
            # 验证参数
            is_valid, errors = self.validator.validate_parameters(
                st.session_state.current_scenario,
                st.session_state.parameters
            )

            if not is_valid:
                st.error("参数验证失败：")
                for error in errors:
                    st.error(f"• {error}")
                return

            # 执行评估
            with st.spinner("正在进行TOPSIS评估..."):
                results = self.evaluator.evaluate(st.session_state.parameters)

            # 显示结果
            self._render_results(results)

        except Exception as e:
            st.error(f"评估过程中发生错误：{str(e)}")

    def _render_results(self, results: Dict[str, Any]):
        """渲染评估结果"""
        st.header("📊 评估结果")

        # 创建多列布局
        col1, col2, col3 = st.columns([1, 1, 1])

        with col1:
            # 综合得分
            st.metric(
                label="综合得分",
                value=f"{results['score']:.3f}",
                delta=None
            )
            st.write(f"**等级**: {results['grade']} ({results['level']})")

        with col2:
            # 相对排名
            st.metric(
                label="相对排名",
                value=f"{results['ranking']}/{results['total_alternatives']}",
                delta=None
            )

        with col3:
            # 得分对比
            current_score = results['score']
            excellent_score = results['detailed_scores']['excellent']
            delta = current_score - excellent_score
            st.metric(
                label="与优秀方案差距",
                value=f"{abs(delta):.3f}",
                delta=f"{delta:+.3f}"
            )

        # 可视化图表
        st.subheader("📈 可视化分析")

        # 创建多行图表
        chart_col1, chart_col2 = st.columns(2)

        with chart_col1:
            # 雷达图
            radar_chart = self.visualizer.create_radar_chart(
                results['indicator_scores'],
                "五大核心指标得分"
            )
            st.plotly_chart(radar_chart, use_container_width=True)

        with chart_col2:
            # 对比图表
            comparison_chart = self.visualizer.create_comparison_chart(results)
            st.plotly_chart(comparison_chart, use_container_width=True)

        # 仪表盘
        gauge_chart = self.visualizer.create_gauge_chart(
            results['score'],
            "综合得分仪表盘"
        )
        st.plotly_chart(gauge_chart, use_container_width=True)

        # 改进建议
        st.subheader("💡 改进建议")

        for suggestion in results['suggestions']:
            st.info(f"• {suggestion}")
```
